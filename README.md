# GPS Location Tracking System using GSM

This project tracks the current location using a GPS module and sends the location details via SMS using a GSM module.

## Tools Used
- GPS Module
- GSM Module
- Arduino
- TinyGPS++ Library

## Working
1. GPS module collects latitude, longitude, speed, and direction.
2. Arduino processes the GPS data.
3. Location details are sent as an SMS through the GSM module.
